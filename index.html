今天有个任务，页面加个访问总次数和在线人数功能

百度完了，再demo上测试了一下发现可行：

上代码：

controller里面：

@Controller
public class PageController {
    @RequestMapping("/session")
    public String session(HttpSession session){
        session.setAttribute("value",111);
        return "login";
    }
}

html页面：

<body style="background-image: url(img/bg1.jpg)">

        <label th:text="${session.value}"></label>
      ...
